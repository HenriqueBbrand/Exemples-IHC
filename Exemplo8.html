Exemplo 8 html - elemento imagem

Para introduzir imagem em nosso documento html usamos a tag img, essa tag possui alguns atributos, tais como src, que representa o caminho da imagem, o metodo width que representa o tamanho da imagem, o atributo height que representa a altura e o atributo alt que é apresentado quando a imagem é quebrada.
