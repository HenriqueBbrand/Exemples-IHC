Exemplo 7 html - elemento html de listas

Listas não ordenadas

Uma lista não ordenada é aquela que apresenta os seus itens por meio de bullets. para representa-la utilizamos as tags ul e li. por exemplo:

  • Item 1
    • SubItem 1.1
    • SubItem 1.2
    • SubItem 1.3
      • Subitem 1.3.1
      • Subitem 1.3.2
      • Subitem 1.3.3
  • Item 2
  • Item 3
  • Item 4
    1. subitem 4.1
    2. subitem 4.2
      • item 4.2.1
      • item 4.2.1
      • item 4.2.1
    3. subitem 4.3

lista ordenada

A lista ordenada é semelhante a lista nao ordenada a diferença é que neste caso os elementos aparecem em ordem numerica

Para representar uma lista ordenada, utilizamos as tags ol e li

por exemplo

  1. Item 1
  2. Item 2
  3. Item 4
  4. item 5
